Portal Maestros y Consultas

Este proyecto se ha desarrollado con el fin de tener un unico interface de entrada de datos de los maestros de las distintas aplicaciones y de esta forma mantener la relacion referencial entre los distintos aplicativos de la empreas. Ademas de utilizar este portal como punto de entrada a un repositorio de distintos informes a peticion de los usuarios, y que en algunos casos pueden necesitar datos de distintas fuentes.

Para desarrollar este proyecto se han utilizado las siguientes tecnologias:

En la capa de frontend, siguiendo patrones Responsive Design, para poder utilizar la aplicacion desde distintos dispositivos se han utilizado las siguintes tecnologias: HTML5, JavaScript, AngularJS y Bootstrap.

Y en la capa de backend se ha utilizado NodeJS, con una base de datos MongoDB para el funcionamiento de la pagina web. Dado que el maestro de los datos esta creado sobre MSSQL Server se crearon distintos servicios web con .Net ( C# ) tanto para realizar las consultas como las operaciones CRUD en la base de datos directamente. Ademas se han programaron una serie de ETL sobre Microsoft que se ejecutaban todas las noches desde los cuales se distribuia la informacion nueva/modificada a trav√©s de los distinos aplicativos.

Para el desarrollo de este portal se han utilizado metodologia SCRUM y Jenkins IC como herramienta de integracion continua.
